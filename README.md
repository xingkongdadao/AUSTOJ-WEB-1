
# AUSTOJ-WEB

使用angular2对[AUSTOJ2][1]项目的前端重构,之前的项目使用Thymeleaf(类似于jsp)后端渲染,维护起来不是很方便.另外个人也想学一门前端框架,因此就有个这个项目.

一些学习经验: [angular2学习记录-给后端程序员的经验分享][2]

因为初学习且前端经验不足,很多东西都不知道该怎么处理合适,所以目前按照自己的想法写,然后一步一步发现缺点再重构,也就是持续学习提升了.
